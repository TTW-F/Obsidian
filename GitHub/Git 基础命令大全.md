## 六、新手必备注意事项

1. 所有命令需在 Git 仓库目录下执行（可通过 `cd 仓库路径` 切换，如 cd D:\Git_Obsidian\Obsidian）；

2. 提交信息（git commit -m ""）需写清晰（如“新增登录按钮样式”），避免无意义内容（如“修改”“更新”）；

3. 推送前建议先拉取（git pull），避免本地版本与远程冲突；

4. GitHub 推送身份验证：HTTPS 方式需用 Personal Access Token（PAT）代替密码，SSH 方式需提前配置密钥；

5. git checkout -- <文件> 会直接丢弃本地修改，且无法恢复，执行前务必确认不需要这些修改。
