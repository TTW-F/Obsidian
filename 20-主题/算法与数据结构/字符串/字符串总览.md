# 字符串总览

## 这是什么

这页用来汇总字符串相关的基础概念、常见操作和高频题型。

字符串表面上像字符数组，但做题时往往不只是“存了一串字符”，还会牵涉匹配、统计、窗口和指针移动。

## 为什么重要

- 很多经典题都发生在字符串上，比如匹配、去重、回文和窗口问题
- 字符串题经常复用数组、哈希表和双指针的方法，是连接多个基础主题的好入口
- 如果只把字符串题当成 API 操作，容易学得很碎

## 我会怎么使用这页

- 遇到字符串题时，先判断它更像匹配问题、统计问题，还是连续区间问题
- 如果发现它本质上是在维护频次或窗口，就从这里跳去相关方法笔记
- 做题复盘时，把“字符串外壳下面真正的结构”补回来

## 易错点

- 容易只盯字符本身，不去看背后是不是数组、哈希表或窗口问题
- 容易把语言里的字符串 API 当成做题思路本身
- 容易忽略大小写、空格、边界和编码这些细节
