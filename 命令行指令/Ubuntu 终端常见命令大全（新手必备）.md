## 六、Ubuntu 终端核心注意事项

1. 【大小写敏感】Ubuntu 终端命令**严格区分大小写**（如 ls、LS、Ls 是三个不同命令），文件名、路径也区分大小写，输入时需准确匹配；

2. 【权限管理】Ubuntu 有严格的权限控制，操作系统级目录（如 /root、/etc）或其他用户的文件时，需加 `sudo` 命令获取管理员权限（如 sudo mkdir /root/Temp），输入时需验证当前用户的密码；

3. 【删除风险】`rm` 命令删除的文件/文件夹无法通过回收站恢复，尤其是 `rm -rf` 命令（强制递归删除），务必确认路径正确（避免误输入 sudo rm -rf /，会删除系统所有文件）；

4. 【路径分隔符】Ubuntu 用 `/` 表示目录分隔符（如 /home/user），与 Windows 的 `\` 相反，注意区分；

5. 【Tab 补全】支持按 Tab 键自动补全命令、文件名、路径（输入前几个字符后按 Tab 即可），可大幅提升输入效率，避免拼写错误；

6. 【命令终止】若命令持续执行（如 ping 未加 -c 参数），可按 `Ctrl+C` 强制终止命令执行。
